⚙️ Installation & Launch

How to install and run Simready Tools in 3ds Max

Method 1: Drag & Drop (Quick)

Step 1

Drag the file

Drag the Nfinite-SimreadyTools.ms file directly into the 3ds Max viewport. The script will automatically register as a macroscript.

Step 2

Assign a hotkey

Open Customize → Customize User Interface → Keyboard. In the Category dropdown, find Nfinite, select SimreadyTools, and assign a convenient shortcut key.

Method 2: Scripting → Run Script

Step 1

Run the script

Open Scripting → Run Script... and select the Nfinite-SimreadyTools.ms file. The main window will appear.

💡 Tip: After the first launch, the script appears in the Nfinite category — you can add it to any toolbar or menu.

🎨 Object Painter

Interactive painting with 3D assets on surfaces — like a brush, but for objects

1. Paint With — Object Source

Source: Selected / List

Where objects come from

Selected — the brush paints with whatever is selected in the viewport when you press START PAINTING. Best for quick work with 1-2 objects.

List — objects are loaded into the Items table using the [+] button. Allows you to prepare a set in advance and change it without restarting.

Pick: Order / Random

Pick order

Order — objects from the list are dispensed in strict sequence (1→2→3→1→2→3...). Ideal for rhythmic placement (benches, lampposts).

Random — each brush click randomly picks an object from the list. Ideal for organic population (books, decor, debris).

Mode: All / Selected

Active filter

All — the brush uses all objects in the Items list.

Selected — only highlighted rows in the Items table participate in painting. Convenient for temporarily limiting the set without removing items from the list.

Items: + / - / X / Sel

List management

[+] — add selected objects to the list. [−] — remove selected rows. [X] — clear the entire list. [Sel] — select the listed objects in the viewport for review.

2. Paint On — Target Surfaces

Target: Grid / Scene / List

Where to paint

Grid — objects are placed on the construction grid. Scene — the brush works on any object in the scene. List — painting only on objects from the Surfaces list.

Elem Only

Element mode

The brush paints only on selected polygons. Select the polygons of a shelf surface → enable Elem Only → paint only on that shelf, ignoring the cabinet body and other surfaces.

💡 Before enabling Elem Only, select the desired polygons in Edit Poly mode on the target object.

3. Settings — Brush Parameters

Spacing

Brush step (cm)

Minimum distance between objects when painting with a stroke. A value of 50 means objects will be placed no more than once every 5 cm.

Offset

Normal offset

Raises or sinks the object relative to the surface. Positive values — the object floats above the surface; negative — it penetrates into it.

Follow

Follow stroke direction

The object faces the direction of brush movement. Ideal for placing objects along a path.

Align / Z-Up / Flip

Orientation

Align — align to the surface normal (object follows the slope).
Z-Up — force the object to stand upright, even on sloped surfaces.
Flip — flip the object upside down (useful for ceiling fixtures).

Auto-Delete Sources

Auto-delete

When enabled, the original source objects are deleted after painting completes. Keeps the scene clean without duplicates.

4. Randomizer — Chaos & Naturalness

X / Y / Z (Position)

Random offset

Adds random positional offset with each click. The value represents the maximum deviation amplitude in centimeters.

Rot X / Rot Y / Rot Z

Random rotation

Enables random rotation around the corresponding axis (0°–360°). Rot Z is the most important button: it rotates the object around the vertical axis, making placement look natural.

💡 For a natural look, enabling only Rot Z is usually enough. Add Rot X/Y only for chaotic scenes (debris, rubble).
Reset Randomizer

Reset

Resets all randomization values to zero.

5. Progress & Controls

Progress Label

Progress counter

Displays "Placed: N / Total" in real time. When all objects are placed, the message "ALL OBJECTS PLACED!" appears.

START PAINTING

Start/Stop toggle

Toggle button. Press once — the brush is active, click on surfaces to paint. Press again or hit Escape — painting ends, temporary groups are dissolved, original names are restored.

📋 Workflow: Object Painter

1. Prepare your assets (product models, decorations, etc.) in the scene.
2. Select Source: List and add models using the [+] button to Items.
3. Select Target: List and add target surfaces (shelves, floor, table).
4. Set Spacing (step distance) and Pick: Random for a natural look.
5. Enable Rot Z in the Randomizer for random rotation.
6. Press START PAINTING and click/stroke across surfaces.
7. Press START PAINTING again or Escape to finish.

🔄 Replacer

Batch replacement of proxy objects with final models, preserving position and hierarchy

Replacement Sources — New Models

New Models List

Replacement list

Add the final models that will replace the proxies. Buttons [+] [−] [X] [Sel] — standard list management.

Pick Mode: Random / Order

Selection mode

Random — each proxy is replaced with a random model from the list. Great for varied population.
Order — models are assigned sequentially with wrapping (1→2→3→1→2→3...).

Targets to Replace — Replacement Targets

Existing Models List

Proxies to replace

Add objects that need to be replaced. Each will be destroyed and a clone from Sources will appear at the same position (with the same transform and parent).

START REPLACEMENT

Execute replacement

Performs batch replacement in a single Undo block. Clones preserve position, rotation, and parent object of the original. A summary report is shown upon completion.

📋 Workflow: Replacer

1. Place simple proxy boxes at the desired positions in the scene.
2. Prepare your final models and add them to Replacement Sources.
3. Select all proxies and add them to Targets to Replace.
4. Choose Random for variety or Order for sequence.
5. Press START REPLACEMENT.

🌿 Scatter

Random distribution of objects across surfaces via raycast

Objects to Scatter — Sources

Items List

Objects to scatter

Add models (grass, rocks, debris, etc.). During scattering, a random object from the list is cloned as an Instance (saving memory).

Surfaces — Target Surfaces

Target Surfaces List

Where to scatter

Surfaces onto which objects will be scattered. Raycasts shoot top-down within the bounding box of all surfaces.

Settings

Count

Number of objects

Target number of objects to place (1–5000). Each object attempts up to 20 raycast tries to find a valid position.

Z Offset

Height offset

Raises or sinks objects in local coordinates. Useful for rocks (−2 cm to partially bury them).

Align to Normal

Surface normal alignment

Objects follow the surface slope (for hills). When disabled, objects always stand upright.

Random Z Rot

Random rotation

Each clone is rotated by a random 0°–360° angle around the Z axis.

Scale Min / Max

Scale range

Random uniform scale in percent (100 = original size). Example: Min 80, Max 120 = 80% to 120% of original size.

📋 Workflow: Scatter

1. Create a terrain surface (Plane, landscape mesh).
2. Prepare scatter objects (rocks, bushes) and add them to Items.
3. Add the terrain to Surfaces.
4. Set Count: 200, enable Align to Normal and Random Z Rot.
5. Set Scale Min: 80, Max: 130 for variation.
6. Press START SCATTER.

📦 Shelf Tool

Automatic product placement on shelves with size calculations, gaps, and interactive Jitter

1. Placement Targets — Shelves

Targets List

Target shelf list

Add one or more shelf objects. The tool processes each shelf sequentially. Supports Editable Poly/Mesh.

Fill All Levels

Auto-detect levels

Scans the cabinet via raycast and finds all horizontal planes (shelves) inside. Each level is filled separately.

Elem Only

Element mode

Fill only selected polygons. Select the polygons of a specific shelf → objects will only be placed there.

2. Objects to Array — Products

Items List

Objects to array

Add product models. The ▲ / ▼ buttons sort by size (smallest to largest and vice versa).

Pattern: Order / Random

Layout pattern

Order — strict product sequence (A-B-C-A-B-C). Random — random pick for each grid cell.

3. Array Settings — Grid

Cols (X) / Rows (Y)

Columns and rows count

Define the array size. Cols = across the shelf width, Rows = along the shelf depth.

Calculate X / Y

Auto-calculate

Automatically calculates how many objects fit on the shelf based on the first item's width and the Gap value.

Depth (Z)

Vertical stacking

Number of "floors" of objects stacked on top of each other. Depth=2 means two stacks.

Gap (cm)

Gap between objects

Spacing between objects in centimeters. Default is 30mm. Increase for loose display, decrease for tight packing.

4. Randomization & Orientation

Pos Jitter / Rot Jitter

Interactive Jitter

Applied in real time to already-placed objects. Change the spinner values and objects move/rotate instantly. Results are deterministic (per-object seed).

+90° / -90° / 180°

Quick rotation

Rotates the entire array by 90° or 180°. Use when products face sideways after Fill Shelf.

5. Actions

FILL SHELF

Fill the shelf

Creates an array of clones (Instance) on each target shelf. Objects are centered on the shelf and distributed across the grid.

UNDO

Undo fill

Deletes all objects from the last fill operation. If pressed again, triggers the standard 3ds Max Undo.

📋 Workflow: Shelf Tool

1. Select the cabinet/shelf and add it to Targets using the [+] button.
2. Add product models to the Items list.
3. Set Gap: 30 and press Calculate X and Calculate Y.
4. Press FILL SHELF.
5. If products face sideways, press +90° or -90°.
6. Adjust Pos Jitter and Rot Jitter for a natural look.
7. Enable Fill All Levels to fill the entire cabinet.

📊 PolyCount

Scene polygon load audit — quickly find the heaviest models

Parameters

Scope: Selected / Scene

Scan scope

Selected — analyzes only selected objects. Scene — scans all objects in the scene.

REFRESH LIST

Refresh data

Recalculates polygon counts and sorts the list from heaviest to lightest. Format: "Polycount | Object Name".

Object List

Interactive selection

Click a row — the object is selected in the viewport. Multi-selection is supported. Use it to quickly navigate to problematic objects.

💡 Tip: Run PolyCount after Scatter/Painter to check if you've overloaded the scene with heavy models. Objects with 50,000+ polygons should be optimized.

💥 MassFX Tools

Create realistic debris piles and heaps using MassFX physics simulation

1. Dynamic Objects — Falling

Items List

Simulation objects

Add objects that will fall and collide. Hierarchies are supported — the script automatically finds all meshes inside.

Mesh Quality

Collision accuracy

Physics mesh resolution (4–256). 128 is the standard for most objects. 256 for complex shapes with cavities (cups, vases). Higher = more accurate but slower.

Material Preset

Physics material

Affects friction and bounce: Cardboard (dull impact), Wood (medium bounce), Rubber (high bounce), Glass (slippery). Choose according to the real-world material.

2. Static Objects — Surfaces

Surfaces List

Immovable objects

Floor, table, shelves — surfaces that dynamic objects will collide with. Uses Original Mesh for precise collisions.

Material Preset

Surface material

Determines surface friction. Concrete (default) — good grip. Ice — slippery.

3. Global Settings — World

Substeps / Solver Iter

Simulation accuracy

Substeps (3) — number of intermediate steps. Increase to 5–10 if objects fall through the floor.
Solver Iter (20) — solver iterations. Increase for dense piles.

Contact Dist / Rest Depth

Contact parameters

Fine-tuning the collision detection distance. Increase Contact Dist if objects interpenetrate.

Use Ground / Alt

Ground plane

Enables an infinite ground plane at height Alt (default 0). Prevents objects from falling into the void.

4. Simulation Controls

PREPARE SIMULATION

Preparation

Adds MassFX_RBody modifiers to all objects. Dynamic = Convex Hull, Static = Original Mesh. Enables control buttons.

Play / Pause

Run physics

Starts the simulation in real time. Press again to pause.

Stop

Capture + Auto Fix

Stops the simulation and automatically fixes hierarchies — _POS helpers are moved to the final mesh positions. This is critical for the Simready pipeline.

Reset

Return to start

Returns all objects to their initial positions (before the simulation started).

Bake

Bake animation

Converts the physics simulation into animation keyframes for rendering.

📋 Workflow: MassFX

1. Add falling objects to Dynamic Objects.
2. Add the floor/table surface to Static Objects.
3. Select the Material Preset (Cardboard for boxes, Wood for planks, etc.).
4. Press PREPARE SIMULATION.
5. Press Play — watch the objects fall.
6. When satisfied with the result, press Stop (auto-fixes hierarchies).
7. If needed, press Bake to save the animation.
8. Press STOP SIMULATION to remove physics modifiers.

📋 Workflow Examples

Typical end-to-end usage scenarios for Simready Tools

🏪 Scenario 1: Populating a Store Shelf

1. Import the cabinet/shelf rack model into the scene.
2. Open Shelf Tool → add the rack to Targets.
3. Import product models → add them to Items.
4. Enable Fill All Levels to fill every shelf.
5. Press Calculate X/Y → FILL SHELF.
6. Rotate products using +90°/-90° if needed.
7. Add Pos Jitter: 0.3 and Rot Jitter: 3 for realism.
8. Use Object Painter (Elem Only) for manual touch-up on individual shelves.

🗑️ Scenario 2: Debris Pile (MassFX)

1. Prepare objects for the pile (boxes, planks, trash).
2. Position them above the target surface (raise by 50–100 cm).
3. Use Scatter to pre-distribute objects above the surface.
4. Open MassFX Tools → add objects to Dynamic, the floor to Static.
5. Select material Cardboard, increase Substeps: 5.
6. PREPARE SIMULATION → Play → Stop when satisfied.
7. Hierarchies are automatically fixed — the scene is ready for export.

🔄 Scenario 3: Replacing Proxies with Final Models

1. Place simple proxy boxes on shelves using Shelf Tool or Painter.
2. Import high-poly final models.
3. Open Replacer → add final models to Sources.
4. Select all proxy boxes → add them to Targets.
5. Select Random → START REPLACEMENT.
6. Verify the result via PolyCount — make sure the scene isn't overloaded.

❓ FAQ — Frequently Asked Questions

Solutions to common problems

Common Questions

Question

Painter doesn't paint — empty clicks

Make sure Target: List is selected and surfaces are added. If Elem Only is enabled, polygons must be pre-selected on the target object.

Question

Shelf Tool places objects sideways

Press +90° or -90° after FILL SHELF. The rotation is applied instantly to the entire array.

Question

MassFX — objects fall through

Increase Substeps to 5-10. Make sure the floor surface is added to Static Objects or Use Ground is enabled.

Question

Scatter doesn't place all objects

Objects find positions via raycast (up to 20 attempts). If the surface is small and Count is large, many may fail to find a spot.

Question

Can I undo actions?

All operations are wrapped in an Undo block. Use Ctrl+Z or the UNDO button in Shelf Tool.
