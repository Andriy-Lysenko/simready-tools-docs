Simready Tools v2.3

Единая профессиональная среда для сборки сцен, наполнения интерьеров и физических симуляций в 3ds Max. Шесть мощных модулей покрывают весь пайплайн — от рисования объектами по поверхностям до физической симуляции завалов.

⬇ Скачать скрипт (.MS) ▶ Видео-туториал

Модули инструмента:

🎨 Object Painter

Интерактивное рисование 3D-ассетами по любым поверхностям с рандомизацией поворота/смещения. Поддержка иерархий и автоудаление источников.

Интерактив

🔄 Replacer

Пакетная замена прокси-объектов на финальные модели с сохранением позиции, поворота и иерархии. Random/Order режимы.

Пакетный

🌿 Scatter

Случайное распределение объектов по поверхностям через рейкаст. Контроль масштаба, поворота и выравнивания по нормали.

Автоматика

📦 Shelf Tool

Автоматическая выкладка товаров на полки с учетом размеров и зазоров. Поддержка многоуровневых шкафов и интерактивный Jitter.

Интерактив

📊 PolyCount

Анализ полигональной нагрузки моделей с сортировкой по тяжести. Быстрый поиск проблемных объектов для оптимизации сцены.

Аналитика

💥 MassFX Tools

Создание реалистичных завалов через физику MassFX. Настройка материалов столкновений, автоматический Bake и фиксация иерархий.

⚙️ Установка и запуск

Как установить и запустить Simready Tools в 3ds Max

Способ 1: Drag & Drop (быстрый)

Шаг 1

Перетащите файл

Перетащите файл Nfinite-SimreadyTools.ms прямо во вьюпорт 3ds Max. Скрипт автоматически зарегистрируется как макроскрипт.

Шаг 2

Назначьте хоткей

Откройте Customize → Customize User Interface → Keyboard. В Category найдите Nfinite, выберите SimreadyTools и назначьте удобную клавишу.

Способ 2: Scripting → Run Script

Шаг 1

Запустите скрипт

Откройте Scripting → Run Script... и выберите файл Nfinite-SimreadyTools.ms. Откроется главное окно.

💡 Совет: После первого запуска скрипт появится в категории Nfinite — его можно добавить на любую панель инструментов или в меню.

🎨 Object Painter

Интерактивное рисование 3D-ассетами по поверхностям — как кисть, но для объектов

1. Paint With — Источник объектов

Source: Selected / List

Откуда берутся объекты

Selected — кисть рисует тем, что выделено во вьюпорте в момент нажатия START PAINTING. Подходит для быстрой работы с 1-2 объектами.

List — объекты загружаются в таблицу Items кнопкой [+]. Позволяет заранее подготовить набор и менять его без перезапуска.

Pick: Order / Random

Порядок выдачи

Order — объекты из списка выдаются строго по очереди (1→2→3→1→2→3...). Идеально для ритмичной расстановки (скамейки, фонари).

Random — каждый клик кисти случайно выбирает объект из списка. Идеально для органичного наполнения (книги, декор, мусор).

Mode: All / Selected

Фильтр активных

All — кисть использует все объекты в списке Items.

Selected — только выделенные строки в таблице Items участвуют в рисовании. Удобно для временного ограничения набора без удаления из списка.

Items: + / - / X / Sel

Управление списком

[+] — добавить выделенные объекты в список. [−] — удалить выделенные строки. [X] — очистить весь список. [Sel] — выделить объекты из списка во вьюпорте для проверки.

2. Paint On — Целевые поверхности

Target: Grid / Scene / List

Куда рисовать

Grid — объекты ставятся на конструкционную сетку. Scene — кисть работает по любому объекту в сцене. List — рисование только по объектам из списка Surfaces.

Elem Only

Режим элементов

Кисть рисует только по выделенным полигонам. Выделите полигоны полки шкафа → включите Elem Only → рисуйте только по этой полке, игнорируя корпус и другие поверхности.

💡 Перед включением Elem Only выделите нужные полигоны в режиме Edit Poly на целевом объекте.

3. Settings — Параметры кисти

Spacing

Шаг кисти (см)

Минимальное расстояние между объектами при рисовании штрихом. Значение 50 = объекты будут ставиться не чаще чем раз в 5 см.

Offset

Смещение по нормали

Поднимает или утапливает объект относительно поверхности. Положительные значения — объект парит над поверхностью, отрицательные — врезается.

Follow

Следование за штрихом

Объект разворачивается «лицом» в направлении движения кисти. Идеально для расстановки объектов вдоль дорожки.

Align / Z-Up / Flip

Ориентация

Align — выравнивать по нормали поверхности (объект следует наклону).
Z-Up — принудительно ставить объект вертикально, даже на наклонной поверхности.
Flip — перевернуть объект (полезно для потолочных светильников).

Auto-Delete Sources

Авто-удаление

Когда включено — оригинальные объекты-источники удаляются после завершения рисования. Сцена остаётся чистой, без дублей.

4. Randomizer — Хаос и естественность

X / Y / Z (Position)

Случайное смещение

Добавляет случайное позиционное смещение при каждом клике. Значение = максимальная амплитуда отклонения в сантиметрах.

Rot X / Rot Y / Rot Z

Случайный поворот

Включает случайный поворот вокруг соответствующей оси (0°–360°). Rot Z — самая важная кнопка: поворачивает объект вокруг вертикальной оси, делая расстановку натуральной.

💡 Для естественного вида достаточно включить только Rot Z. Добавляйте Rot X/Y только для хаотичных сцен (мусор, обломки).
Reset Randomizer

Сброс

Обнуляет все значения рандомизации до нуля.

5. Прогресс и управление

Progress Label

Счётчик прогресса

Показывает «Placed: N / Total» в реальном времени. Когда все объекты размещены — появляется сообщение «ALL OBJECTS PLACED!».

START PAINTING

Запуск/остановка

Кнопка-переключатель. Нажмите — кисть активна, кликайте по поверхностям. Нажмите ещё раз или Escape — рисование завершается, группы разбираются, имена восстанавливаются.

📋 Рабочий процесс: Object Painter

1. Подготовьте ассеты (модели товаров, декора и т.д.) в сцене.
2. Выберите Source: List и добавьте модели кнопкой [+] в Items.
3. Выберите Target: List и добавьте целевые поверхности (полки, пол, стол).
4. Настройте Spacing (шаг) и Pick: Random для естественного вида.
5. Включите Rot Z в Randomizer для случайного поворота.
6. Нажмите START PAINTING и кликайте/проводите кистью по поверхностям.
7. Нажмите START PAINTING повторно или Escape для завершения.

🔄 Replacer

Пакетная замена прокси-объектов на финальные модели с сохранением позиции и иерархии

Replacement Sources — Новые модели

New Models List

Список замен

Добавьте финальные модели, которые заменят прокси. Кнопки [+] [−] [X] [Sel] — стандартное управление списком.

Pick Mode: Random / Order

Режим выбора

Random — каждый прокси заменяется случайной моделью из списка. Подходит для вариативного наполнения.
Order — модели назначаются по порядку с цикличностью (1→2→3→1→2→3...).

Targets to Replace — Цели замены

Existing Models List

Прокси для замены

Добавьте объекты, которые нужно заменить. Каждый будет уничтожен и на его месте (с тем же transform и parent) появится клон из Sources.

START REPLACEMENT

Запуск замены

Выполняет пакетную замену в одном блоке Undo. Клоны сохраняют позицию, поворот и родительский объект оригинала. По завершении — отчёт о количестве заменённых объектов.

📋 Рабочий процесс: Replacer

1. Расставьте простые прокси-боксы в нужных позициях сцены.
2. Подготовьте финальные модели и добавьте их в Replacement Sources.
3. Выделите все прокси и добавьте в Targets to Replace.
4. Выберите Random для вариативности или Order для порядка.
5. Нажмите START REPLACEMENT.

🌿 Scatter

Случайное распределение объектов по поверхностям через рейкаст

Objects to Scatter — Источники

Items List

Объекты для разброса

Добавьте модели (трава, камни, мусор и т.д.). При скаттере случайный объект из списка клонируется как Instance (экономия памяти).

Surfaces — Целевые поверхности

Target Surfaces List

Куда раскидать

Поверхности, по которым будут разбросаны объекты. Рейкаст стреляет сверху вниз в пределах bounding box всех поверхностей.

Settings — Настройки

Count

Количество объектов

Целевое число объектов для размещения (1–5000). Каждый объект ищет валидную позицию до 20 попыток рейкаста.

Z Offset

Смещение по высоте

Приподнимает или утапливает в локальных координатах. Полезно для камней (−2 см, чтобы частично врыть).

Align to Normal

По нормали поверхности

Объект повторяет наклон поверхности (для холмов). Если выключено — всегда стоит вертикально.

Random Z Rot

Случайный поворот

Каждый клон поворачивается на случайный угол 0°–360° вокруг Z.

Scale Min / Max

Диапазон масштаба

Случайный uniform-масштаб в процентах (100 = оригинал). Пример: Min 80, Max 120 = от 80% до 120% размера.

📋 Рабочий процесс: Scatter

1. Создайте ландшафтную поверхность (Plane, terrain).
2. Подготовьте объекты (камни, кусты) и добавьте в Items.
3. Добавьте ландшафт в Surfaces.
4. Укажите Count: 200, включите Align to Normal и Random Z Rot.
5. Задайте Scale Min: 80, Max: 130 для вариативности.
6. Нажмите START SCATTER.

📦 Shelf Tool

Автоматическая выкладка товаров на полки с учётом размеров, зазоров и интерактивным Jitter

1. Placement Targets — Полки

Targets List

Список целевых полок

Добавьте один или несколько объектов-полок. Инструмент обработает каждую полку последовательно. Поддерживаются Editable Poly/Mesh.

Fill All Levels

Автопоиск уровней

Сканирует шкаф рейкастом и находит все горизонтальные плоскости (полки) внутри. Каждый уровень заполняется отдельно.

Elem Only

Режим элементов

Заполнение только по выделенным полигонам. Выделите полигоны одной полки → объекты встанут только туда.

2. Objects to Array — Товары

Items List

Объекты для выкладки

Добавьте модели товаров. Кнопки ▲ / ▼ сортируют по размеру (от маленьких к большим и наоборот).

Pattern: Order / Random

Паттерн выкладки

Order — строгая последовательность товаров (A-B-C-A-B-C). Random — случайный выбор для каждой ячейки.

3. Array Settings — Сетка

Cols (X) / Rows (Y)

Количество колонок и рядов

Определяют размер массива. Cols = по ширине полки, Rows = в глубину полки.

Calculate X / Y

Авторасчёт

Автоматически вычисляет количество объектов, помещающихся на полку, исходя из ширины первого объекта в Items и значения Gap.

Depth (Z)

Стопка в высоту

Количество «этажей» объектов друг на друге. Depth=2 = две стопки.

Gap (cm)

Зазор между объектами

Расстояние между объектами в сантиметрах. По умолчанию 30мм. Увеличьте для свободной выкладки, уменьшите для плотной.

4. Randomization & Orientation

Pos Jitter / Rot Jitter

Интерактивный Jitter

Применяется в реальном времени к уже расставленным объектам. Меняйте значения спиннеров — объекты двигаются/поворачиваются мгновенно. Результат детерминирован (seed на каждый объект).

+90° / -90° / 180°

Быстрый доворот

Разворачивает весь массив на 90° или 180°. Используйте, если товары стоят боком после Fill Shelf.

5. Действия

FILL SHELF

Заполнить полку

Создаёт массив клонов (Instance) на каждой целевой полке. Объекты центрируются на полке и распределяются по сетке.

UNDO

Отмена заполнения

Удаляет все объекты последнего заполнения. Если нажать повторно — вызывает стандартный Undo 3ds Max.

📋 Рабочий процесс: Shelf Tool

1. Выделите шкаф/полку и добавьте в Targets кнопкой [+].
2. Добавьте товары в список Items.
3. Установите Gap: 30 и нажмите Calculate X и Calculate Y.
4. Нажмите FILL SHELF.
5. Если товары стоят боком — нажмите +90° или -90°.
6. Покрутите Pos Jitter и Rot Jitter для натуральности.
7. Включите Fill All Levels если нужно заполнить весь шкаф.

📊 PolyCount

Аудит полигональной нагрузки сцены — быстрый поиск тяжёлых моделей

Параметры

Scope: Selected / Scene

Область сканирования

Selected — анализирует только выделенные объекты. Scene — сканирует все объекты в сцене.

REFRESH LIST

Обновить данные

Пересчитывает полигоны и сортирует список от тяжёлых к лёгким. Формат: «Polycount | Object Name».

Список объектов

Интерактивный выбор

Клик по строке — объект выделяется во вьюпорте. Мульти-выбор поддерживается. Используйте для быстрого перехода к проблемным объектам.

💡 Совет: Запускайте PolyCount после Scatter/Painter — проверьте, не перегрузили ли вы сцену тяжёлыми моделями. Объекты с 50 000+ полигонов стоит оптимизировать.

💥 MassFX Tools

Создание реалистичных завалов и куч через физическую симуляцию MassFX

1. Dynamic Objects — Падающие

Items List

Объекты для симуляции

Добавьте объекты, которые будут падать и сталкиваться. Поддерживаются иерархии — скрипт автоматически находит все меши внутри.

Mesh Quality

Точность коллизий

Разрешение физической сетки (4–256). 128 — стандарт для большинства объектов. 256 — для сложных форм с полостями (чашки, вазы). Чем выше — тем точнее, но медленнее.

Material Preset

Физический материал

Влияет на трение и отскок: Cardboard (глухой удар), Wood (средний отскок), Rubber (сильный отскок), Glass (скольжение). Выбирайте соответственно реальному материалу.

2. Static Objects — Поверхности

Surfaces List

Неподвижные объекты

Пол, стол, полки — поверхности, о которые динамические объекты будут ударяться. Используется Original Mesh для точных коллизий.

Material Preset

Материал поверхности

Определяет трение поверхности. Concrete (по умолчанию) — хорошее сцепление. Ice — скольжение.

3. Global Settings — Мир

Substeps / Solver Iter

Точность симуляции

Substeps (3) — количество промежуточных шагов. Увеличьте до 5–10 если объекты проваливаются сквозь пол.
Solver Iter (20) — итерации солвера. Увеличьте для плотных завалов.

Contact Dist / Rest Depth

Параметры контакта

Тонкая настройка дистанции обнаружения столкновений. Increase Contact Dist если объекты проникают друг в друга.

Use Ground / Alt

Плоскость пола

Включает бесконечную плоскость-пол на высоте Alt (по умолчанию 0). Предотвращает падение в бесконечность.

4. Управление симуляцией

PREPARE SIMULATION

Подготовка

Добавляет MassFX_RBody модификаторы ко всем объектам. Dynamic = Convex Hull, Static = Original Mesh. Активирует кнопки управления.

Play / Pause

Запуск физики

Запускает симуляцию в реальном времени. Нажмите повторно для паузы.

Stop

Фиксация + Auto Fix

Останавливает симуляцию и автоматически исправляет иерархии — _POS хелперы перемещаются к финальным позициям мешей. Это критично для Simready пайплайна.

Reset

Возврат к началу

Возвращает все объекты в исходные позиции (до начала симуляции).

Bake

Запечь анимацию

Конвертирует физическую симуляцию в ключи анимации для рендеринга.

📋 Рабочий процесс: MassFX

1. Добавьте падающие объекты в Dynamic Objects.
2. Добавьте поверхность пола/стола в Static Objects.
3. Выберите Material Preset (Cardboard для коробок, Wood для досок и т.д.).
4. Нажмите PREPARE SIMULATION.
5. Нажмите Play — наблюдайте за падением.
6. Когда результат устроит — нажмите Stop (авто-фиксация иерархий).
7. При необходимости нажмите Bake для сохранения анимации.
8. Нажмите STOP SIMULATION для удаления физических модификаторов.

📋 Примеры рабочих процессов

Типичные сценарии использования Simready Tools от начала до конца

🏪 Сценарий 1: Наполнение магазинной полки

1. Импортируйте модель шкафа/стеллажа в сцену.
2. Откройте Shelf Tool → добавьте стеллаж в Targets.
3. Импортируйте товары → добавьте в Items.
4. Включите Fill All Levels для заполнения всех полок.
5. Нажмите Calculate X/Y → FILL SHELF.
6. Доверните товары кнопками +90°/-90° если нужно.
7. Добавьте Pos Jitter: 0.3 и Rot Jitter: 3 для реалистичности.
8. Используйте Object Painter (Elem Only) для ручной доработки отдельных полок.

🗑️ Сценарий 2: Завал обломков (MassFX)

1. Подготовьте объекты для завала (коробки, доски, мусор).
2. Расположите их над целевой поверхностью (приподнимите на 50–100 см).
3. Используйте Scatter для предварительного разброса объектов над поверхностью.
4. Откройте MassFX Tools → добавьте объекты в Dynamic, пол в Static.
5. Выберите материал Cardboard, увеличьте Substeps: 5.
6. PREPARE SIMULATION → Play → Stop когда устроит.
7. Иерархии автоматически зафиксированы — сцена готова к экспорту.

🔄 Сценарий 3: Замена прокси на финальные модели

1. Расставьте простые прокси-боксы на полках с помощью Shelf Tool или Painter.
2. Импортируйте финальные high-poly модели.
3. Откройте Replacer → добавьте финальные модели в Sources.
4. Выделите все прокси-боксы → добавьте в Targets.
5. Выберите Random → START REPLACEMENT.
6. Проверьте результат через PolyCount — убедитесь, что сцена не перегружена.

❓ FAQ — Частые вопросы

Решения типичных проблем

Общие вопросы

Вопрос

Painter не рисует — пустой клик

Убедитесь, что Target: List выбран и поверхности добавлены. Если используется Elem Only — полигоны должны быть предварительно выделены на целевом объекте.

Вопрос

Shelf Tool ставит объекты боком

Нажмите +90° или -90° после FILL SHELF. Доворот применяется мгновенно ко всему массиву.

Вопрос

MassFX — объекты проваливаются

Увеличьте Substeps до 5-10. Убедитесь, что поверхность пола добавлена в Static Objects или включён Use Ground.

Вопрос

Scatter не размещает все объекты

Объекты ищут позицию через рейкаст (до 20 попыток). Если поверхность маленькая, а Count большой — многие могут не найти места.

Вопрос

Можно ли отменить действие?

Все операции оборачиваются в блок Undo. Используйте Ctrl+Z или кнопку UNDO в Shelf Tool.
